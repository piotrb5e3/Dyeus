# Dyeus
Dyeus is a web application for gathering and displaying sensory data, and making recommendations based on preset rules.

#Usage
### Requirements
* Git
* Python 3

### Setup
* Clone this repository
* Optionally create a virtualenv
* `cd Dyeus`
* `pip install -e .`

### Development run
`pserve development.ini --reload`

### Running tests
`pytest`
